# SouthScipy-Cafe
**##Description**

SouthScipy Cafe is a modern, AI-powered cafe web application that provides customers with a seamless online dining experience. Users can browse the menu, order food online, reserve tables, interact with an AI chatbot, receive personalized food recommendations, and manage their accounts through a responsive and user-friendly interface.
 ## Features
🍽️ Online Food Ordering
📅 Table Reservation System
👤 User Registration & Login
🤖 AI-Powered Food Recommendation System
💬 AI Chatbot for Customer Support
⭐ Customer Reviews & Ratings
📱 Fully Responsive Design
🎨 Modern and Attractive User Interface
🔍 Search & Filter Menu Items
📞 Contact & Support
⚡ Fast and Smooth Navigation
🌐 Cross-Browser Compatibility
## Technologies Used
HTML5
CSS3
JavaScript
React.js
Node.js
Express.js
MongoDB
AI Recommendation System
Chatbot Integration
## Project Objective

The objective of this project is to build a smart cafe management platform that enhances the customer experience by combining online ordering, table booking, AI-powered recommendations, and chatbot assistance in one application.
